Agilidade

O sistema foi pensado para agilizar a consulta dos processos de auxílios estudantis da UFPB, diminuindo o tempo de pesquisa e facilitando o acesso a qualquer estudante.

Entenda os processos

De segunda a sexta, em horário comercial, é coletado as informações do site do SIPAC. Essas informações são salvas em nosso banco de dados e estão disponíveis na forma de API Rest, podendo ser consumida por qualquer sistema. Para saber mais, acesse aqui.
